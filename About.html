Le HyperText Markup Language, généralement abrégé HTML ou, dans sa dernière version, HTML5, est le langage de balisage conçu pour écrire les pages web.

Il s'agit d'un format ouvert très utilisé en informatique.Comme son nom l'indique, HTML permet d’écrire de l’hypertexte.
